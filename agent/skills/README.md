# Agent Skills

This directory contains "skills" that extend the capabilities of your AI agent.

## Adding a New Skill

1. Create a new directory for your skill: `mkdir my-skill`
2. Create a `SKILL.md` inside it.
3. Follow the [Agent Skills Specification](https://agentskills.io/specification).

## Installation

These skills are automatically symlinked to `~/.agent/skills` by the dotfiles setup script.
